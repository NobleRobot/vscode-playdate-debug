# VS Code Playdate Debug

Unofficial Playdate debug extension

## Configuration

```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "type": "playdate",
      "request": "launch",
      "name": "Debug (Simulator)",
      "sdkPath": "/path/to/PlaydateSDK",
      "sourcePath": "/path/to/MyGame/source",
      "gamePath": "/path/to/MyGame/MyGame.pdx"
    }
  ]
}
```
